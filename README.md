# Portfolio professionnel

Portfolio statique en francais, pense pour presenter le parcours professionnel de Corentin Webber avec une structure orientee recrutement.

## Modifier le contenu

- `index.html` : intro, chiffres cles, experiences, a propos, methode, realisations, competences, formation, langues/outils, passions et contact.
- `styles.css` : couleurs, mise en page, responsive.
- `script.js` : menu mobile et annee automatique.
- `assets/cv-corentin-webber.pdf` : CV ajoute au site.
- `assets/corentin-webber-photo.jpeg` : photo professionnelle utilisee dans l'intro.

## Mettre en ligne

Ce site ne demande aucune compilation. Il peut etre publie directement sur GitHub Pages, Netlify, Vercel ou n'importe quel hebergement statique.

Avant diffusion, verifier :

- les formulations que tu veux rendre plus personnelles ;
- les chiffres et dates avant publication.
